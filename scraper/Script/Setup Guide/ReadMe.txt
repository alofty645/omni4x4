1) Download and install Python from this link: https://www.python.org/downloads/
2) Install python libraries listed in requirements.txt file
   Use this command in Terminal/Command Prompt/Shell: pip install library_name
3) Download chromedriver according to your OS and Chrome Version(Check in Settings > About) from 
   this link: https://chromedriver.chromium.org/downloads
4) Unzip chromedriver file. Replace the old chromedriver.exe file with this one.
5) Run the script: 
   - Open project folder in Command Prompt/Shell/Terminal
   - run this command: python 4wdsupacentre.py
6) A browser will open and the output 4wdsupacentre.csv file will be generated in the same folder
